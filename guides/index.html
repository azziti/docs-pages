Guides

Overview

Lumio is a lightweight task runner for defining and executing automated workflows. It lets you chain tasks, handle dependencies, and run steps in parallel or sequentially from a single configuration file.

Lumio is designed to be embedded in any project regardless of language or framework. It has zero runtime dependencies and runs as a standalone CLI or as a Node.js library.

Installation

Install Lumio via npm:

npm install @lumio/runner

Or install the CLI globally:

npm install -g lumio

Node.js 18 or later is required.

Usage

Configuration file

Create a lumio.config.ts at the root of your project:

import { defineConfig } from "@lumio/runner";

export default defineConfig({
  tasks: {
    lint: {
      run: "eslint .",
    },
    test: {
      run: "jest --coverage",
      dependsOn: ["lint"],
    },
    build: {
      run: "tsc",
      dependsOn: ["test"],
    },
  },
});

Running tasks

Run a single task:

lumio run build

Run all tasks in dependency order:

lumio run --all

Run tasks in parallel where dependencies allow:

lumio run --all --parallel

Programmatic usage

Use Lumio directly in your Node.js scripts:

import { createRunner } from "@lumio/runner";
import config from "./lumio.config";

const runner = createRunner(config);
await runner.run("build");
